Servicios Rest:
El proyecto rest está dentro del directorio rest-api del repo, acá la instrucciones para correrlo:
    -npm install (instalar las dependencias)
    -node index.js (levanta los servicios rest, Server listening on port : 3000)

Proyecto Swagger:
El proyecto swagger está dentro del directorio swagger-server, para levantarlo:
    -npm install (instalar las dependencias)
    -node index.js (levanta el servidor Swagger, Swagger-ui is available on http://localhost:8080/docs )
