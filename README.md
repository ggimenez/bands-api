# bands-api

Node.js REST API of music rock bands

## Usage

### Install:

    $ npm install  //to install the dependencies
    
### Run server:
    $ npm start  //at running at http://localhost:3000

### Testing:
    $ npm test 

### Swagger docs
    at running at http://localhost:3000/api-doc   

## Requirements

* Node v0.10.32 or above
